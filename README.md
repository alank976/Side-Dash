# Side Dash

Web UI for measurement of capacities spent on side tasks

## What is "Side task"?
Works related to business but not rioritized stories, i.e. not new features delivery. E.g. support issue, enquiries, hardware/software upgrade, interview, meetings, 

## How to start it up?
1. Create an environment by:
`conda create -n <env_name>`
2. Install all required dependencies: `pip install -r requirements.txt`
3. `python app.py` to serve it
